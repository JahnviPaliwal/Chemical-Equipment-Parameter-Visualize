
#  Chemical Equipment Parameter Visualizer

### Web Application & Desktop Application (Electron)

A **full-stack data visualization and report generation system** for chemical equipment datasets.
The application is available as:

*  **Web Application** (React + Django)
*  **Desktop Application** built using **Electron**

Users can upload CSV files, visualize insights (tables, charts, summaries), and generate **PDF reports that preserve the exact frontend layout**.
The backend stores datasets and frontend-generated PDFs while maintaining a history of the **latest 5 uploads**.

---

##  Key Features

* Upload CSV files containing chemical equipment data
* Automatic backend analysis after upload
* Interactive frontend visualization (charts, tables, summaries)
* **Frontend-generated PDF reports (charts + UI preserved)**
* Backend stores exact PDFs generated by frontend
* History panel showing last 5 uploads with downloadable reports
* Desktop application using **Electron**
* Single backend shared by web & desktop apps

##  Web Application Usage

The web version runs in the browser and communicates with the Django backend via REST APIs.

### Web App Flow:

1. User uploads CSV
2. Backend analyzes data
3. Frontend renders charts & tables
4. User clicks **Download Report**
5. PDF is generated on frontend and saved to backend
6. Report appears in History section

---

##  Desktop Application (Electron)

The **desktop app wraps the same React frontend** using Electron, allowing the application to run as a native desktop application on:

* Windows
* macOS
* Linux

### Desktop App Highlights:

* Same UI as web app
* Uses same Django backend
* Offline-ready UI (backend required for uploads)
* Native window controls

##  Backend Setup (Django)

```bash
cd backend
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Backend URL:

```
http://127.0.0.1:8000/
```

---

##  Frontend Setup (React)

```bash
cd frontend
npm install
npm start
```

Frontend URL:

```
http://localhost:3000/
```

##  Electron Desktop App Setup

### Install Dependencies

```bash
cd electron
npm install
```

### Start Desktop App (Development)

```bash
npm start
```

> Electron loads the React frontend and connects to the same Django backend.

##  PDF Report Handling (Important Design)

* PDFs are generated **only on the frontend**
* Backend **never regenerates** reports
* Backend stores the **exact frontend-rendered PDF**
* Charts, layout, tables, and formatting remain unchanged

This ensures **true WYSIWYG reporting**.

---

##  API Endpoints

| Method | Endpoint        | Description                   |
| ------ | --------------- | ----------------------------- |
| POST   | `/api/upload/`  | Upload CSV                    |
| PUT    | `/api/upload/`  | Upload frontend-generated PDF |
| GET    | `/api/summary/` | Latest dataset summary        |
| GET    | `/api/history/` | Last 5 uploads                |
| GET    | `/media/<file>` | Download stored PDF           |
